{% trans "Rules" %}

{% trans "Everything you need to know to play Pixo Canvo." %}

{% blocktrans trimmed %} Pixo Canvo is a real-time collaborative canvas. Each registered player can place one pixel every 30 seconds. Every placement is queued, validated, and then broadcast live via WebSocket to every connected player. Work together — or compete — to claim space on the shared canvas. {% endblocktrans %}

🎨 {% trans "Placing pixels" %}

  1. 1

    {% trans "Pick a color from the palette" %}

    {% blocktrans trimmed %}Choose from the 21-color palette at the top of the Canvas page. Use the Custom Color picker next to the palette for any hex color you like. Your active selection is highlighted with a ring.{% endblocktrans %}

  2. 2

    {% trans "Zoom and navigate" %}

    {% blocktrans trimmed %}Use the + and − buttons to zoom. Pan by scrolling or dragging. The ⛶ button enters full-screen mode for a distraction-free experience. Zoom in before placing to avoid missing your target cell.{% endblocktrans %}

  3. 3

    {% trans "Click to place" %}

    {% trans "Click any cell on the canvas while a color is selected. Your placement is sent to the server, validated (cooldown, bounds, auth), and if accepted it appears on every connected player's screen within milliseconds." %}

  4. 4

    {% trans "Wait out the cooldown" %}

    {% blocktrans trimmed %}After each accepted placement you must wait 30 seconds before placing again. The status bar shows how long remains. Rejected attempts do not reset the cooldown.{% endblocktrans %}

  5. 5

    {% trans "Highlight your pixels" %}

    {% blocktrans trimmed %}Press Highlight My Pixels to dim everything else and see exactly which pixels you own on the current canvas.{% endblocktrans %}

📐 {% trans "Dynamic canvas size" %}

{% blocktrans trimmed %} The canvas starts at a fixed size and automatically grows as more pixels are placed. Once 80 % of cells are filled the grid expands by a set number of rows and columns, up to a configured maximum. This keeps the canvas competitive — there is always new space to claim. {% endblocktrans %}

🏘 {% trans "Communities" %}

{% blocktrans trimmed %} Beyond the global canvas you can play inside private communities. Each community has its own independent canvas, leaderboard, and chat room. Go to the Communities page to manage yours. {% endblocktrans %}
✏️

{% trans "Creating a community" %}

{% blocktrans trimmed %}Any logged-in user can create communities without a hard limit. Each new community costs 60 Pixo. Enter a name and click Create community. A URL-friendly slug is generated automatically. You become the owner and are added as the first member.{% endblocktrans %}

🔗

{% trans "Inviting members" %}

{% blocktrans trimmed %}Click Invitation link on any community card to open a popup with a shareable URL. Anyone who opens that link and is logged in will be added to the community immediately.{% endblocktrans %}

🚪

{% trans "Leaving a community" %}

{% blocktrans trimmed %}Members can leave any community they did not create. Click Leave community and confirm. The owner cannot leave — they must delete the community instead.{% endblocktrans %}

🗑️

{% trans "Deleting a community" %}

{% blocktrans trimmed %}Only the owner can delete a community. Click Delete community and confirm. This permanently removes the community, all its pixels, chat messages, and membership records — it cannot be undone.{% endblocktrans %}

📊

{% trans "Community details page" %}

{% blocktrans trimmed %}Click Community details on any card to open a dedicated page showing: community info, a live canvas preview (click to enlarge), and a sortable member table with each member's join date and pixel count.{% endblocktrans %}

🏆 {% trans "Leaderboard & chat" %}

🥇

{% trans "Ranking" %}

{% blocktrans trimmed %}The Ranking page ranks every contributor by the number of accepted pixels currently on the canvas. The list is available for the global canvas and for each community you belong to.{% endblocktrans %}
